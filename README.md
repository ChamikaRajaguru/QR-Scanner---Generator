# 📱 QR Scanner & Generator (Flutter)

A simple and efficient QR Code Scanner and Generator mobile application built using Flutter. This app allows users to scan QR codes in real-time and generate custom QR codes instantly with a smooth and user-friendly interface.

---

## 🚀 Features

- 📷 Real-time QR Code scanning using camera
- 🔳 Generate QR codes instantly
- 📋 Display scanned results clearly
- 🔗 Supports URLs, text, and other QR formats
- 🎨 Clean and intuitive user interface
- ⚡ Fast and responsive performance

---

## 🛠️ Built With

- Flutter (Dart)
- Camera / QR scanning libraries (e.g., mobile_scanner / qr_code_scanner)
- QR code generation libraries
